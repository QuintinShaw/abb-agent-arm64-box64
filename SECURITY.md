# Security Policy

[English](SECURITY.md) | [中文](SECURITY.zh-CN.md)

Report security issues, legal concerns, or infringement requests to:

`github@xyt.email`

Do not open public issues containing:

- NAS usernames or passwords
- Synology account credentials
- Tokens, session IDs, or certificates
- Private NAS hostnames or internal domains
- Complete ABB logs containing identifiable server data
- Generated packages containing Synology proprietary binaries

Before sharing logs, redact NAS addresses, account names, tokens, UUIDs that
identify your device or NAS, certificates, and internal network details.

This project is in beta testing and is not supported by Synology. Do not use it
as the only backup path for important data. Always validate restores before
trusting any backup system.
